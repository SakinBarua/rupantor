# রূপান্তর (RUPANTOR) — Premium Card Game Website

> কার্ড নয়, পৃথিবী বদলাও

একটি সম্পূর্ণ ফাংশনাল অ্যাডভান্সড প্রিমিয়াম ওয়েবসাইট — HTML, CSS, JS দিয়ে তৈরি।

## 📁 ফাইল স্ট্রাকচার

```
rupantor-website/
├── index.html              ← হোম পেজ
├── css/
│   ├── style.css           ← সম্পূর্ণ ডিজাইন সিস্টেম
│   ├── game.css            ← গেম-নির্দিষ্ট স্টাইল
│   └── responsive.css      ← 🆕 রেসপন্সিভ + টিউটোরিয়াল + স্মুথনেস
├── js/
│   ├── data.js             ← ২৫০ কার্ড + ২০০ মিশন + ১০ এক্সপানশন ডাটা
│   ├── main.js             ← সমস্ত ইন্টারঅ্যাকটিভ ফিচার
│   ├── tutorial.js         ← 🆕 ৭-ধাপ ইন্টারঅ্যাকটিভ টিউটোরিয়াল
│   └── game.js             ← সম্পূর্ণ প্লে-যোগ্য গেম লজিক
├── pages/
│   ├── cards.html          ← কার্ড ভাণ্ডার (ফিল্টার + সার্চ + মডাল)
│   ├── rulebook.html       ← রুলবুক (ট্যাব-ভিত্তিক)
│   ├── play.html           ← 🎮 ইন্টারঅ্যাকটিভ গেম (প্লেয়ার vs AI)
│   ├── missions.html       ← ২০০ গোপন মিশন
│   ├── expansions.html     ← ১০ এক্সপানশন প্যাক
│   └── faq.html            ← প্রশ্নোত্তর (অ্যাকর্ডিয়ন)
├── assets/
│   └── images/             ← (খালি — SVG ইনলাইন ব্যবহৃত)
└── README.md               ← এই ফাইল
```

## ✨ ফিচার

### হোম পেজ (`index.html`)
- প্রিমিয়াম হিরো সেকশন পার্টিকেল অ্যানিমেশন সহ
- অ্যানিমেটেড কাউন্টার (২৫০ কার্ড, ২০০ মিশন, ১০০ প্রতীক)
- গেম দর্শন (৬টি ফিচার কার্ড)
- কার্ড প্রিভিউ (৮টি র‍্যান্ডম কার্ড)
- খেলার নিয়ম (৪-ধাপ)
- ১৪ কার্ড শ্রেণীর গ্রিড
- CTA সেকশন

### কার্ড ভাণ্ডার (`pages/cards.html`)
- ২৫০টি কার্ড সম্পূর্ণ গ্রিডে
- **১৪ শ্রেণীর ফিল্টার** বোতাম
- **সার্চ বক্স** (নাম বা নাম্বার দিয়ে)
- প্রতিটি কার্ডে ক্লিক করলে **ডিটেইল মডাল** খোলে
- প্রোগ্রাম্যাটিক SVG আর্ট প্রতিটি কার্ডে
- রিয়েল-টাইম কাউন্ট আপডেট

### রুলবুক (`pages/rulebook.html`)
- ৫টি ট্যাব: সেটআপ, টার্ন কাঠামো, ক্রিয়া, খেলোয়াড় মোড, উন্নত
- প্রতিটি সেকশনে আইকন ও বিস্তারিত বিবরণ
- সব মোড: Solo, 2-6P, Tournament, Campaign, Hardcore, Family, Kids

### গোপন মিশন (`pages/missions.html`)
- ২০০টি মিশন সম্পূর্ণ গ্রিডে
- প্রতিটির নাম্বার, নাম, বিবরণ, পুরস্কার
- হোভার ইফেক্ট সহ

### এক্সপানশন (`pages/expansions.html`)
- ১০টি এক্সপানশন প্যাক
- প্রতিটির কার্ড ও মিশন সংখ্যা
- প্রকাশের সময়সূচি

### প্রশ্নোত্তর (`pages/faq.html`)
- অ্যাকর্ডিয়ন স্টাইল FAQ
- ক্লিক করে উত্তর দেখুন/লুকান

### 🎮 খেলুন (`pages/play.html`) — সম্পূর্ণ প্লে-যোগ্য গেম
- **প্লেয়ার বনাম AI** কার্ড স্থাপন গেম
- **৩টি কঠিনতা স্তর**: সহজ, সাধারণ, কঠিন
- **৮×৬ গ্রিড টেবিল** — কার্ড স্থাপনের জন্য
- **৫টি কার্ডের হাত** — ২৫০ কার্ড থেকে এলোমেলো
- **দিক নির্বাচন** — উত্তর/পূর্ব/দক্ষিণ/পশ্চিম
- **সংযোগ সনাক্তকরণ** — সংলগ্ন কার্ডের স্বয়ংক্রিয় সংযোগ
- **অনুরণিত সংযোগ** — দিক মিললে বোনাস পয়েন্ট
- **গোপন মিশন** — ৬টি মিশন থেকে ২টি এলোমেলো
- **রিয়েল-টাইম স্কোরিং** — কার্ড, সংযোগ, মিশন পয়েন্ট
- **গেম লগ** — প্রতিটি চালের বিবরণ
- **টোস্ট নোটিফিকেশন** — মিশন পূরণ, সতর্কতা
- **বিজয় পর্দা** — চূড়ান্ত স্কোর ও পরিসংখ্যান
- **AI কৌশল** — কঠিন মোডে AI কৌশলগতভাবে খেলে

### 📚 টিউটোরিয়াল সিস্টেম (`js/tutorial.js`)
- **৭টি ইন্টারঅ্যাকটিভ ধাপ** — স্বাগতম, হাত, স্থাপন, দিক, সংযোগ, মিশন, বিজয়
- **প্রতিটি ধাপে ভিজ্যুয়াল ডেমো** — SVG ও CSS অ্যানিমেশন
- **স্কিপ বোতাম** — যেকোনো সময় টিউটোরিয়াল স্কিপ করুন
- **প্রগ্রেস ইন্ডিকেটর** — বর্তমান ধাপ দেখায়
- **পরবর্তী/আগের নেভিগেশন** — কীবোর্ড সাপোর্ট (← → ESC)
- **প্রথমবার স্বয়ংক্রিয়** — নতুন ব্যবহারকারীর জন্য স্বয়ংক্রিয় আবির্ভাব
- **পুনরায় দেখুন** — স্টার্ট স্ক্রিন থেকে যেকোনো সময়

### 📱 সম্পূর্ণ রেসপন্সিভ (`css/responsive.css`)
- **৫টি ব্রেকপয়েন্ট**: 480px, 768px, 900px, 1024px, 1440px+
- **মোবাইল-ফার্স্ট ডিজাইন** — ছোট স্ক্রিনে সম্পূর্ণ ব্যবহারযোগ্য
- **টাচ-অপ্টিমাইজড** — ট্যাপ টার্গেট ৪৪px+, স্ক্রল অপ্টিমাইজেশন
- **স্মুথ অ্যানিমেশন** — cubic-bezier ইজিং, হার্ডওয়্যার অ্যাক্সেলারেশন
- **অ্যাক্সেসিবিলিটি** — reduced-motion সাপোর্ট, focus states, ARIA

## 🚀 কীভাবে চালাবেন

1. ZIP ফাইল এক্সট্র্যাক্ট করুন
2. `index.html` ফাইলটি যেকোনো ব্রাউজারে খুলুন (Chrome, Firefox, Safari, Edge)
3. অথবা লোকাল সার্ভার চালান:

```bash
# Python দিয়ে
python3 -m http.server 8000

# Node দিয়ে
npx serve
```

তারপর ব্রাউজারে যান: `http://localhost:8000`
